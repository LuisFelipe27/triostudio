# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Python 3   | :white_check_mark: |
| Python 2   | :x:                |
| Django <= 3.2   | :white_check_mark: |
| Django > 3.2   | :x:                |

## Reporting a Vulnerability

Always check the versions of the libraries in pipfile.
